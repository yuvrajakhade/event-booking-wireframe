# event-management-wireframe (Expo UI)

Static UI prototype in **React Native (Expo + TypeScript)** using **React Navigation**.

## Screens

- Booked Events (date filter + search + Add)
- Enquiries (date filter + search)
- Event Add/Edit (static form)
- Check-In (static list)
- Check-Out (computes missing inventory)
- Missing Inventory (details)

## Install & run

> You already have `package.json` committed on `expo-ui`.

```bash
npm install
npm run start
```

## Notes

- This is **static UI** using mock data in `src/data/mock.ts`
- No backend, no persistence
